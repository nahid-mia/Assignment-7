## This is a [Next.js](https://nextjs.org) project "KeenKeeper"

This app here helps bring kins closer. Helping and tracking and keeping all the records here for a mutual interaction ultimately makes life log more easier. Call, Video and Text options are all integrated for a user so that a user doesn't need to go many places to do those different things.


## Technologies Used
1. Next.js framework
2. React
3. Javascript
4. Tailwind
5. DaisyUI
6. Toaster
7. Recharts



## Features
1. Calling, Texting, And videoCalling options are side by side here.
2. All the logs are kept for a user.
3. THe logs can be filtered and seen with exact time and date of happening.
